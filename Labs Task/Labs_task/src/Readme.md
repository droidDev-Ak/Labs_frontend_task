Product Management Dashboard

This project is a frontend assignment built as part of an interview process. It demonstrates a simple product management interface with core features such as listing, searching, adding, editing, and paginating products, all handled on the client side.

Live Demo

Deployed Application:


- Features

Product list displayed in both table view and card (grid) view

Toggle option to switch between table and card layouts

Real-time product search by name with 500ms debounce

Add new products using a validated form

Edit existing products with clear visual indication

Client-side pagination for product listing

Fully responsive and user-friendly interface

- Tech Stack

React

JavaScript

Tailwind CSS

Notes and Assumptions

Product data is stored and managed entirely in memory

No backend or external API is used

Sample product data is included to allow immediate interaction with the application

- Running the Project Locally

Clone the repository

Install dependencies using npm install

Start the development server using npm run dev

Open the application in the browser
